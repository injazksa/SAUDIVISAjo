### 🌐 الموقع الرسمي
المركز المعتمد من السفارة السعودية لإنجاز تأشيرات العمل

### 📋 المحتوى
- **587+ مهنة معتمدة** مع الأوراق المطلوبة لكل مهنة
- **تصديق الشهادات** من السفارة السعودية
- **الاعتماد المهني السعودي (إنجاز)**
- **حاسبة الرسوم** التقريبية
- **مدونة** مع مقالات مفيدة

### 🚀 التقنيات المستخدمة
- HTML5, CSS3, JavaScript (Vanilla)
- Tailwind CSS (via CDN)
- Font Awesome Icons
- Google Fonts (Alexandria)

- `certificates.html` - تصديق الشهادات
- `professional.html` - الاعتماد المهني
- `styles.css` - التصميم الكامل
- `script.js` - الوظائف

### 🔧 نشر الموقع على Netlify

#### الخطوة 1: رفع على GitHub
```bash
git add .
git commit -m "Complete website files"
git push origin main
```

#### الخطوة 2: ربط Netlify
1. سجل دخول على [Netlify](https://netlify.com)
2. اضغط "Add new site" → "Import an existing project"
3. اختر GitHub واختر repository: `SAUDIVISAjo`
4. الإعدادات:
   - **Branch:** `main`
   - **Publish directory:** `.` (نقطة فقط)
   - **Build command:** اتركه فاضي
5. اضغط "Deploy"

#### Netlify CMS (اختياري)
الموقع جاهز لـ Netlify CMS:
- الملفات في: `/admin/`
- بعد النشر، ادخل على: `your-site.netlify.app/admin/`
