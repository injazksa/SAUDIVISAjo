# مكتب تأشيرات السعودية - دليل الاستخدام

## ربط نموذج الاتصال بالإيميل

لربط نموذج الاتصال بإيميلك الشخصي، اتبع الخطوات التالية:

### الخيار 1: استخدام Formspree (مجاني وسهل)

1. قم بزيارة موقع [Formspree](https://formspree.io/)
2. قم بإنشاء حساب مجاني
3. أنشئ نموذج جديد (New Form)
4. أدخل بريدك الإلكتروني: `info.injaz.sa@gmail.com`
5. ستحصل على رابط مثل: `https://formspree.io/f/xxxxxxxx`
6. افتح ملف `index.html` وابحث عن السطر:
   ```html
   <form class="contact-form" id="contactForm" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
7. استبدل `YOUR_FORM_ID` بالرمز الذي حصلت عليه من Formspree

### الخيار 2: استخدام EmailJS

1. قم بزيارة موقع [EmailJS](https://www.emailjs.com/)
2. قم بإنشاء حساب مجاني
3. اتبع التعليمات لربط حسابك بإيميلك
4. احصل على Service ID, Template ID, و Public Key
5. أضف الكود التالي في ملف `script.js`

### الخيار 3: استخدام خادم خاص (Backend)

إذا كنت تملك خادم خاص، يمكنك إنشاء API endpoint يستقبل البيانات ويرسلها عبر البريد الإلكتروني.

## ملاحظات مهمة

- النموذج الحالي يعرض رسالة نجاح عند الإرسال
- لتفعيل إرسال الإيميل الفعلي، يجب استخدام أحد الخيارات المذكورة أعلاه
- جميع الخيارات المذكورة توفر خطط مجانية كافية لمعظم الاستخدامات
