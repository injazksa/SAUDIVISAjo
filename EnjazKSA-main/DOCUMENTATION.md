## 💼 إدارة المهن

### إضافة مهنة جديدة

1. اذهب إلى `/admin`
2. اختر **"المهن والأوراق المطلوبة"**
3. انقر **"New Profession"**
4. أدخل البيانات التالية:

| الحقل | الوصف | مثال |
|------|-------|------|
| كود المهنة | رمز المهنة الرسمي | 2141 |
| اسم المهنة | اسم المهنة بالعربية | مهندس مدني عام |
| التصنيف | فئة المهنة | هندسية |
| التكلفة | السعر التقريبي (اختياري) | 500 |
| الملاحظات | معلومات إضافية (اختياري) | - |

5. انقر **"Publish"**

### تعديل مهنة موجودة

1. اذهب إلى `/admin`
2. اختر **"المهن والأوراق المطلوبة"**
3. اختر المهنة من القائمة
4. عدّل البيانات المطلوبة
5. انقر **"Publish"**

### حذف مهنة

1. اختر المهنة من القائمة
2. انقر على **"Delete"** (إذا كانت متاحة)
3. أكد الحذف

### البحث عن مهنة

استخدم شريط البحث في لوحة التحكم للعثور على مهنة معينة:
- ابحث بالاسم: "مهندس"
- ابحث بالكود: "2141"

## 📝 إدارة المدونة

### إضافة مقالة جديدة

1. اذهب إلى `/admin`
2. اختر **"المدونة (المقالات)"**
3. انقر **"New Post"**
4. أدخل البيانات:

| الحقل | الوصف | مثال |
|------|-------|------|
| العنوان | عنوان المقالة | دليل شامل للتأشيرات |
| الرابط الفريد | slug بدون مسافات | comprehensive-visa-guide |
| التاريخ | تاريخ النشر | 2026-02-24 |
| الصورة | صورة الغلاف (800x400) | - |
| الوصف | نص قصير للـ SEO | وصف المقالة هنا... |
| الكاتب | اسم الكاتب (اختياري) | أحمد محمد |
| المحتوى | نص المقالة بصيغة Markdown | - |

5. انقر **"Publish"**

### تنسيق المقالات (Markdown)

استخدم Markdown لتنسيق النصوص:

```markdown
# العنوان الرئيسي
## العنوان الفرعي
### عنوان صغير

**نص غامق**
*نص مائل*

- نقطة 1
- نقطة 2
- نقطة 3

1. نقطة مرقمة
2. نقطة مرقمة

[رابط](https://example.com)

![صورة](https://example.com/image.jpg)
```

### نشر المقالة

بعد النقر على "Publish":
- سيتم حفظ المقالة تلقائياً في GitHub
- ستظهر على الموقع خلال دقائق
- ستُفهرس من قبل محركات البحث

- ✅ تشفير البيانات المرسلة
- ✅ منع الوصول للملفات الحساسة

1. **استخدم كلمات مرور قوية** على GitHub

## 🔍 استكشاف الأخطاء

### المشكلة: لا تظهر التحديثات على الموقع

**الحل:**
1. تحقق من أن التحديث تم نشره ("Publish")
2. امسح ذاكرة التخزين المؤقت للمتصفح (Ctrl+Shift+Delete)
3. انتظر 5 دقائق لنشر التحديث
4. تحقق من سجل النشر على Netlify

### المشكلة: الصور لا تظهر

**الحل:**
1. تحقق من أن الصورة تم رفعها بنجاح
2. تأكد من صيغة الصورة (PNG, JPG, WebP)
3. تحقق من حجم الصورة (< 5MB)
4. استخدم صور بدقة عالية (1920x1080 أو أكثر)

### المشكلة: البحث لا يعمل بشكل صحيح

**الحل:**
1. تحقق من أن البيانات محملة بشكل صحيح
2. امسح ذاكرة التخزين المؤقت
3. أعد تحميل الصفحة (F5)
4. جرب البحث بكلمات مختلفة

### المشكلة: لا يمكن تسجيل الدخول إلى لوحة التحكم

**الحل:**
1. تأكد من أن حسابك على GitHub نشط
2. تحقق من أن لديك صلاحيات الوصول للمستودع
3. امسح ملفات تعريف الارتباط (Cookies)
4. جرب متصفح مختلف
